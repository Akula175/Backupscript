## Description: 

## The backup project will consist of a script that backs up data in a directory on a local computer or a remote computer accessed via SSH.

## The script will take as argument; a directory, or IP:Port/directory in case of remote backup.



## Features:

## Scheduler; backups should be scheduled to be run ahead of time.

## Restore; script should be able to restore backups to the original destination. 

## Restore test; script should test restores by restoring to a test directory.

## Remote; script should be able to back up via SSH remotely.

## Encryption; script should support encryption
